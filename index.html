Projects

Discriminative Learning Approaches for Face Verification
The Siamese architecture is used for training a similarity metric from data. The method is used for recognition or verification applications where the number of classes is very large and not known during training, and where the number of training samples for a single class is very small and the class is not important. The idea is to learn a function that maps input patterns into a target space with the impostor pairs(pairs which do not belong to the same class) become as separated as possible and genuine pairs(pairs which belong to the same class) become as close as possible using the simple distance metrics. The project is aimed to implement discriminating algorithm for face verification applications. Tensorflow has been used as the deep learning framework.
QR Code Pattern Recognition and Message Extraction
The goal of this project is to successfully detect and reconstruct perfect QR-code pattern and then decode and extract the message and information within. Usually the QR-code images are corrupted, Blurred or at least rotated which make the pattern recognition harder than simple scenarios. In this situation robust algorithms can effectively recognize specfic patterns in the image and reconstruct the main matrix of quick response code. In the first part of this project the implemented software, which is developed using MATLAB, successfully recognize the main QR pattern and then extract the QR-Matrix. Then by using decoding techniques, the message is extracted from the QR-code. Note that this project is restricted to QR codes version-1 through version-6..
Create LMDB file from Numpy -- A Pythonic Approach for Creating Caffe DATA files
Caffe as a deep learning framework supports different file formats. LMDB and HDF5 are the ones which are of great interest. Caffe users usually prefer LMDB due to speed and memory matters. However there is a necessity to transform usual file formats to the formats adaptable by the Caffe framework. Numpy is a famous package by python which makes it very simple to create ".npy" file from features which have to be used for machine learning and classification purposes. This project is aimed to transform a pack of numpy files to a single LMDB file.

Local Histogram Equalization
The project aim is to locally equalize the histogram of an image which has been proven to be more effective than the normal Histogram Localization. By the adjusments made by Histogram Equalization the intensities would be better distributed on the histogram which led to specific areas of lower local contrast gain a higher contrast. Parallel computing is used to accelerating the process. OpenCV, the famous Computer Vision open-source library has been implemented as a dependency.
Siamese Convolutional Neural Network
The project is aimed to implement the Siamese architecture using Convolutional Neural Network. TensorFlow has been used as the deep learning framework. The approach is to learn a nonlinear mapping that learns a similarity metric from the data itself for face verification which suggests a more data-driven solution compared to the traditional facial texture analysis methods.
