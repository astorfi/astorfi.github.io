# astorfi.github.io

Personal website of Sina Torfi — Senior Applied Scientist at Amazon.

Built with pure HTML, CSS, and JavaScript. No build tools or frameworks required.

## Pages

- **Home** — Hero, stats, experience preview, skills, selected publications
- **Experience** — Full work timeline and education
- **Projects** — Research and applied AI project showcase
- **Publications** — Featured papers with citation counts
- **Repositories** — Open-source projects and educational resources

## Local Development

Open `index.html` in a browser, or serve locally:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
